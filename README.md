### 1\. Run Inference from Command Line

Try our example inference script. 
```bash
bash inference.sh
```


**Inference settings:**

  * `--deepseek_api_key`: Put your Deepseek API key here. We are sorry that due to the principle of anonymity, we are unable to provide you with the available api (Default: `None`)
  * `--key_prompt "a park"`: Put your scene text that you want. (Default: `a park`)
